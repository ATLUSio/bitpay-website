Improving the bitcoin protocol, bit by bit.

BIPs are proposals to improve the bitcoin protocol itself - we foster and encourage contributions to the bitcoin protocol. Here are a few we have written.

Deterministic Multisignature Wallets

BIP 0045

This BIP defines a structure for hierarchical deterministic P2SH multi-party multi-signature wallets

Add Mempool Message Proposal

BIP 0035

Make a network node's transaction memory pool accessible via a new "mempool" message.

HTTP Payment Request Proposal

BIP 0073

A standard spec for requesting and recurring signal payent request from an HTTP server.
